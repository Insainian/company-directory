# Company Directory

## Initial Setup

```bash
npm install
```

## Run Server

```bash
npm run start
```

## Development

For incremental builds and hot reload,
run the following first:


```bash
npm run webpack
```

Then run the following in another terminal:

```bash
npm run start:webpack
```

## Technology Stack

### Frontend
